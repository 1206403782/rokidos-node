# node-ams

Rokid Application Node.js API library.

### Installation

> System builtin module

Or via NPM

```sh
$ npm install @rokid/ams
```

### Usage

```js
const app = require('@rokid/ams')();
app.on('request', function(result) {
  // event trigger
});
app.start();
```
